title: "Goals Toolkit：Huberman 的 13 个目标设定工具"
description: "Andrew Huberman 总结的 13 个循证目标设定工具，涵盖目标选择、视觉化、奖励机制与昼夜节律。"
---


- Tool 1: Choose a Priority Goal
- Tool 2: Pursue Lofty Goal **选难的目标，才能触发大脑可塑性，但不能太难，倒不能完成的难度。**
- Tool 3: Define Verb Actions, Measurability & Specificity; Writing vs. Typing
- Tool 4: Visual Reminder Myth; “Post-It Fallacy” **简单来说，没用**
- Tool 5: Accountability Myth, “Don’t Tell the World” Rule
- Tool 6: Measurable Goal; Quarterly Cycle
- Tool 7: Quantifiable Goals; Book Writing **不能量化的目标，应该怎么办 actions and actions themselves are always quantifiable it's number of hours of dedicated work toward that particular goal。
- Tool 8: Visualization of End; Motivation & Negative Thinking **想象自己在这十二周的所有努力和付出之后失败会如何。（ 设目标的时候就应该不光只想成功，还有为了这个需要的付出）
- Tool 9: Visual Target/Finish Line Training & Perceived Effort
- Tool 10: Distance from Phone **简单来说：关机**
- Tool 11: Random, Intermittent Reinforcement; Cognitive Rewards **赌场策略，奖励有时候给，有时候不给**
- Tool 12: “Middle Problem”; Time Chunking  
- Tool 13: Circadian Rhythm & Attention **一天三个时间最有能量：醒来之后30分钟，3个小时，11个小时** [[日程表]]
