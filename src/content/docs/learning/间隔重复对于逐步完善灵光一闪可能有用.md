在谈话或散步中间，我经常灵光一现，想出有趣的问题和想法。大部分时候，我当场是没办法围绕冒出的想法写出很有见地的东西的。我自然写不出高质量的[常青笔记](https://link.zhihu.com/?target=https%3A//notes.andymatuschak.org/z4SDCZQeRo4xFEQ8H4qrSqd68ucpgE6LU155C)<sup data-text="常青笔记 (Evergreen notes)" data-url="https://zhuanlan.zhihu.com/p/416319260" data-numero="1" data-draft-node="inline" data-draft-type="reference" data-tooltip="常青笔记 (Evergreen notes) <a href=&quot;https://zhuanlan.zhihu.com/p/416319260&quot; rel=&quot;noopener noreferrer&quot; target=&quot;_blank&quot;>https://zhuanlan.zhihu.com/p/416319260</a>" data-tooltip-richtext="1" data-tooltip-preset="white" data-tooltip-classname="ztext-reference-tooltip"><a id="ref_1_0" href="https://zhuanlan.zhihu.com/p/#ref_1" data-reference-link="true" aria-labelledby="ref_1">[1]</a></sup>。对于这个想法我还无可置喙——只是它看起来特别有趣。

我现在应该怎么做？我应该如何安排行动，从而逐步完善这个灵光一闪？我可以创建一个「待办事项」，或者安排一段时间来思考这个问题，但完善灵感往往不该这么做。相反，我常常该让子弹飞一会：过几天再回顾，看看我又想出了什么。

我可以在[为临时性的和不完整的笔记设置写作收件箱](https://link.zhihu.com/?target=https%3A//notes.andymatuschak.org/z5aJUJcSbxuQxzHr2YvaY4cX5TuvLQT7r27Dz)<sup data-text="为临时性的和不完整的笔记设置写作收件箱" data-url="https://zhuanlan.zhihu.com/p/442485060" data-numero="2" data-draft-node="inline" data-draft-type="reference" data-tooltip="为临时性的和不完整的笔记设置写作收件箱 <a href=&quot;https://zhuanlan.zhihu.com/p/442485060&quot; rel=&quot;noopener noreferrer&quot; target=&quot;_blank&quot;>https://zhuanlan.zhihu.com/p/442485060</a>" data-tooltip-richtext="1" data-tooltip-preset="white" data-tooltip-classname="ztext-reference-tooltip"><a id="ref_2_0" href="https://zhuanlan.zhihu.com/p/#ref_2" data-reference-link="true" aria-labelledby="ref_2">[2]</a></sup>中记录这个概念，但收件箱中的内容会迅速变成一堆难以处理的碎片，而我会忽略这些碎片 ([收件箱只有在你相信它们是如何被清空的情况下才有效](https://link.zhihu.com/?target=https%3A//notes.andymatuschak.org/z5tiFxnNKMZCnc8G9R1N51L5hknyRGmyCQx18)<sup data-text="收件箱需要可靠的清空流程才有效" data-url="https://zhuanlan.zhihu.com/p/434717044" data-numero="3" data-draft-node="inline" data-draft-type="reference" data-tooltip="收件箱需要可靠的清空流程才有效 <a href=&quot;https://zhuanlan.zhihu.com/p/434717044&quot; rel=&quot;noopener noreferrer&quot; target=&quot;_blank&quot;>https://zhuanlan.zhihu.com/p/434717044</a>" data-tooltip-richtext="1" data-tooltip-preset="white" data-tooltip-classname="ztext-reference-tooltip"><a id="ref_3_0" href="https://zhuanlan.zhihu.com/p/#ref_3" data-reference-link="true" aria-labelledby="ref_3">[3]</a></sup>)。

[间隔重复系统可以用来编程注意力](https://link.zhihu.com/?target=https%3A//notes.andymatuschak.org/z2gqazXUkf9qyFjMQg4W3dw6yegnAJszvDywN)<sup data-text="可使用间隔重复系统对注意力编程" data-url="https://zhuanlan.zhihu.com/p/414293765" data-numero="4" data-draft-node="inline" data-draft-type="reference" data-tooltip="可使用间隔重复系统对注意力编程 <a href=&quot;https://zhuanlan.zhihu.com/p/414293765&quot; rel=&quot;noopener noreferrer&quot; target=&quot;_blank&quot;>https://zhuanlan.zhihu.com/p/414293765</a>" data-tooltip-richtext="1" data-tooltip-preset="white" data-tooltip-classname="ztext-reference-tooltip"><a id="ref_4_0" href="https://zhuanlan.zhihu.com/p/#ref_4" data-reference-link="true" aria-labelledby="ref_4">[4]</a></sup>，所以间隔重复在这里可能会有帮助。在这样一个系统中，我可能会：

- 坐下来进行晨间写作，阅读当天的写作卡片，这些卡片从写作收件箱中抽取，数量很少
- 我可以添加我目前对卡片内容的想法（也许这些想法让以前写的东西意图更模糊了，除非我添加新的材料）
- 完成之后，我就可以把卡片标记为「有成果」，这意味着它很快就会再次出现；或者标记成「无成果」，系统便大大增加其间隔时间。
- 另外，我可能会花点时间把一些卡片转换成一条或多条[常青笔记](https://link.zhihu.com/?target=https%3A//notes.andymatuschak.org/z4SDCZQeRo4xFEQ8H4qrSqd68ucpgE6LU155C)<sup data-text="常青笔记 (Evergreen notes)" data-url="https://zhuanlan.zhihu.com/p/416319260" data-numero="1" data-draft-node="inline" data-draft-type="reference" data-tooltip="常青笔记 (Evergreen notes) <a href=&quot;https://zhuanlan.zhihu.com/p/416319260&quot; rel=&quot;noopener noreferrer&quot; target=&quot;_blank&quot;>https://zhuanlan.zhihu.com/p/416319260</a>" data-tooltip-richtext="1" data-tooltip-preset="white" data-tooltip-classname="ztext-reference-tooltip"><a id="ref_1_1" href="https://zhuanlan.zhihu.com/p/#ref_1" data-reference-link="true" aria-labelledby="ref_1">[1]</a></sup>。
- 任何我直接忽略的卡片，其间隔时间都会增加，但增加的间隔时间比起显然无用的卡片少。

利用间隔重复的指数性质，上百张卡片逐步推进便成为可能，而每天只消考虑其中的几张。

这将会构成「渐进思考」的系统。
