## 两种笔记

### 来源型笔记
- [[灵感笔记]]  fleeting note 
- [[文献笔记]]
需要经常及时整理，最好是**每天整理。**
这两种笔记是想法和知识的来源。
阅读这两种笔记，然后去思考，新记的这些内容，和旧的卡片之间可能有什么联系。

### 永久笔记
- [[永久笔记]] 
这种笔记是思考的过程和结果。
对比新旧内容，有什么新的发现，产生了什么新的问题。


把[[卡片盒笔记法]] 变成了维基百科，就缺失了最关键的一步，**思考**被丢失了。

看似凌乱，但在凌乱中催生着规律，这个规律就是自己的[[洞见 insight]]

1. 写灵感笔记
2. 对比旧的永久笔记
3. 思考有什么新的发现
4. 丢掉灵感笔记


### 输出 output

- 在形成聚集效应的地方，**新的主题**可能已经产生。



排除所有干扰之后，思考才是唯一有意义的事。[[思考的意义]]
